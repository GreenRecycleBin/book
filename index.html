Welcome!

Welcome to smlhelp.github.io! The goal of this site is to provide a convenient resource for learning functional programming, particularly in the language of Standard ML. You'll find various guides and references about the SML type system & functional programming concepts, as well as some worked examples and tutorials. Click on the navigation links to get started!
